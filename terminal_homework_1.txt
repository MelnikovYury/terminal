Первая часть первого ДЗ 
Linux terminal (GitBash) commands

1. Посмотреть где я - pwd
2. Создать папку - mkdir folder_1
3. Зайти в папку - cd folder_1
4. Создать 3 папки - mkdir folder_1_1 folder_1_2 folder_1_3
5. Зайти в любую папку - cd folder_1_1
6. Создать 5 файлов (3 txt, 2 json) - touch file1.txt file2.txt file3.txt file4.json file5.json
7. Создать 3 папки - mkdir folder_txt folder_json folder_html
8. Вывести список содержимого папки - ls -la
9. Открыть любой txt файл - nano file1.txt
10. Написать туда что-нибудь, любой текст - Пишем произвольный текст
11. Сохранить и выйти - ctrl+O, ctrl +X
12. Выйти из папки на уровне выше - cd ..
13. Переместить любые 2 файла, которые вы создали, в любую другую папку - mv folder_1_1/file1.txt folder_1_2/  
14. Скопировать любые 2 файла, которые вы создали, в любую другую папку - cp folder_1_1/file2.txt folder_1_3/file2copy.txt
15. Найти файл по имени - find ~/Desktop/folder_1 -name "file1.txt"
16. Посмотреть содержимое в реальном времени - tail -f ../folder_1_2/file1.txt
17. Вывести несколько первых строк из текстового файла - head -n2 ../folder_1_2/file1.txt
18. Вывести несколько последних строк из текстового файла - tail -n2 ../folder_1_2/file1.txt
19. Посмотреть содержимое длинного файла - less ../folder_1_2/file1.txt
20. Вывести дату и время - date
